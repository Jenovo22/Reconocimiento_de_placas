# Imágenes procesadas

Esta carpeta almacena imágenes generadas por la aplicación después del procesamiento.

Cada imagen procesada puede incluir:

- Cajas de detección de placas.
- Confianza de detección.
- Texto OCR reconocido.
- Confianza del OCR.

Estos archivos son generados automáticamente y normalmente no deben subirse al repositorio.
