# Videos procesados

Esta carpeta almacena videos generados por la aplicación después del procesamiento.

Cada video procesado puede incluir:

- Detección de placas.
- Seguimiento con ID por placa.
- Mejor confianza histórica por ID.
- Mejor OCR histórico por ID.
- Exportación en formato MP4 compatible con navegador.

Estos archivos son generados automáticamente y normalmente no deben subirse al repositorio.
